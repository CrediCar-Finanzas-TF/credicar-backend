# CrediCar Backend

## Summary
This repository contains the backend for the CrediCar application, built with Java, Spring Boot, and Spring Data JPA, using a MySQL database. It includes interactive API documentation through Swagger UI and practically implements key financial concepts (such as loan amortization, VAN, and TIR) learned in the *Finance and Economic Engineering* course.


## Features
- RESTful API
- OpenAPI Documentation
- Swagger UI
- Spring Boot Framework
- Spring Data JPA
- Validation
- MySQL Database
- Domain-Driven Design
